# Building

`mvn clean package`

# Running

`java -jar target/udemy-reviews-producer-1.0-SNAPSHOT-jar-with-dependencies.jar`

With custom course ID
`COURSE_ID=1141696 java -jar target/udemy-reviews-producer-1.0-SNAPSHOT-jar-with-dependencies.jar`
